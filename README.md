# Sistema de estoque de produtos
## função
- Adicionar o Produto
- Listar todos os produtos disponíveis 
- Atualizar preço e quantidade do produto
- Excluir produto
- Buscar produtos da tabela




# Como usar?
## DOCS
1 - entre no .venv
- Instale os requirements.txt no terminal bash

pip install dontev fastapi psycopg2-binary psycopg2 uvicorn


entre na pasta back-end
- rode o main.py no terminal

uvicorn main:app --reload



## Site STREAMLIT
entre na pasta front end
- rode o app.py no terminal

python -m streamlit run app.py
